  ## Installation
  
- Download The Repo
- Run "npm i" in terminal
  
  ## Usage
  
- You can opt to run "npm run seed" to seed the database.
- Run "npm start" to run the backend
- In Insomnia you can run GET, POST, PUT, DELETE to modify the data
